RedFlag-AI is an AI-powered assistant designed to help emergency departments
triage patients by identifying medical red flags and classifying severity
levels (GRIS, VERT, JAUNE, ROUGE).

## Objective
Provide fast, explainable, and reliable patient triage support using
medical rules, machine learning, and LLM-based explanations.

## Scope
This project focuses on **emergency patient triage**.

## Tech Stack
- Python
- Machine Learning
- LLM
- RAG
